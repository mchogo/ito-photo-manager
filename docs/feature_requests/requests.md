# いとうさんフォトマネージャー 改修・機能追加要望メモ

本ドキュメントは、いとうさんフォトマネージャーに対する今後の改修および機能追加の要望をまとめたものです。

## 1. Kintone連携関連
- **Note出力機能**: Kintone上で管理しているデータから、Note（ノート）を出力できるようにする。

## 2. LINE WORKS連携
- **自動登録・通知**: 可能であれば、指定のタイミング（例：情報の更新時や写真のアップロード時など）で、LINE WORKSに自動で情報が登録・通知される機能を実装する。

## 3. 帳票・ドキュメント出力およびファイルサーバー連携
- **完成図書のアップロード**: 完成図書一式を所定のファイルサーバーへ直接アップロードできるようにする。
- **出力対象および構成**:
  - 報告書
  - 写真シート（複数枚のシート出力に対応すること）
  - 依頼シート

## 4. 入館・アクセス管理（検討事項）
- **QRコード入館**: 現場等の入館処理において、QRコードを利用したスムーズな入館受付機能の導入を検討する。
